# ⚡ Quick Start : Activer le Design System Dynamique

## 🎯 En 3 minutes chrono

### 1️⃣ Ouvrez l'interface admin
```
Connectez-vous → Admin → Design System
```

### 2️⃣ Cliquez sur "Copier le script SQL"
Un bouton apparaît avec ce message :
```
❌ Design system non initialisé
📋 Cliquez sur "Copier le script SQL"
```

### 3️⃣ Ouvrez Supabase Dashboard
```
https://supabase.com/dashboard/project/[VOTRE_PROJECT_ID]
→ SQL Editor
→ New Query
→ Collez le script (Ctrl+V)
→ Run (ou F5)
```

### 4️⃣ Retournez dans l'app
```
Cliquez sur "Vérifier à nouveau"
```

### 5️⃣ Confirmez le succès
Vous devriez voir :
```
✅ Design system dynamique opérationnel
✅ La table design_tokens existe et contient un thème actif
```

## 🎨 C'est prêt !

Vous pouvez maintenant :
- Modifier les couleurs en temps réel
- Ajuster la typographie globale
- Personnaliser l'apparence complète de GasyWay
